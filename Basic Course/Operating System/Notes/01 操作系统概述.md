## 复习：理解计算机硬件 (电路)

前导知识：数字逻辑电路/计算机系统基础

-   一个极简的公理系统 (导线、时钟、逻辑门、触发器)
-   建立在公理体系上的数字系统设计 (包括计算机)
    -   前导课程目标：能根据需求实现数字系统

### Logisim Demo

数字电路模拟器

-   基本构件：wire, reg, NAND, NOT, AND, NOR
-   每一个时钟周期
    -   先计算 wire 的值
    -   在周期结束时把值锁存至 reg

会编程，你就拥有全世界！

-   同样的方式可以模拟任何数字系统 (包括计算机系统)
-   同时还体验了 UNIX 哲学
    -   Make each program do one thing well
    -   Expect the output of every program to become the input to another
